    GIT Branch erstellen


    Befehl Beschreibung
    git branch test Erzeugt einen neuen branch mit dem namen test
    git checkout test Zum test branch wechseln bzw. master
    git checkout -b err001 Erzeugt einen neuen branch namens err001 und wechselt zum branch
    git merge hotfix Branch zusammenführen
    git branch -d hotfix Entfernt den erstellten branch
    git branch -v Letzten commit aller branches anzeigen
